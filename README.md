## 🎯 What This Project Does

RetailSense AI bridges the gap between raw business data and actionable decisions. Instead of writing SQL or waiting for a data team, any stakeholder can type a natural language question and receive:

- ✅ An **auto-generated SQL query** (transparent and reviewable)
- ✅ An **interactive chart** (auto-selected based on data shape)
- ✅ A **downloadable data table**
- ✅ A **plain-English business insight** with an actionable recommendation generated by Gemini AI

## 🚀 How to Run Locally

### 1. Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/retailsense-ai.git
cd retailsense-ai
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Add your Gemini API Key
Get a free key at [aistudio.google.com](https://aistudio.google.com) — no credit card needed.

Open `app.py` and replace line 8:
```python
API_KEY = "your-gemini-api-key-here"
```

### 4. Set up the database
```bash
python setup_db.py
```

### 5. Run the app
```bash
streamlit run app.py
```

Open your browser at `http://localhost:8501`

## 🔑 Key Features

### 🤖 Natural Language to SQL
The LLM is prompted with the full table schema, so it generates accurate, SQLite-compatible queries. Column names with spaces are handled correctly. Date functions use SQLite's `strftime()`.

### 📊 Smart Chart Selection
The app auto-selects the best chart type:
- **Line chart** for time-series / trend questions
- **Bar chart** for category comparisons (≤10 items)
- **Horizontal bar** for ranked lists (>10 items)
- **Scatter plot** for two-variable comparisons

### 🧠 Executive Insight Generation
After fetching data, a second LLM call generates a 3–5 sentence business insight; written for non-technical stakeholders, with one actionable recommendation.

### 📥 Data Export
Every result can be downloaded as a CSV for further analysis.

## 📊 Dataset

The dataset simulates a real retail business with:
- **2,000 orders** across 2021–2023
- **4 regions** (West, East, Central, South)
- **3 customer segments** (Consumer, Corporate, Home Office)
- **3 product categories** (Furniture, Office Supplies, Technology)
- **15 sub-categories** and 60+ products
- **Metrics**: Sales, Profit, Quantity, Discount per order
